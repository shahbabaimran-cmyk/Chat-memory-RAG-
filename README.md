# Chat-memory-RAG-
# 🧠 Conversational RAG Memory System

A lightweight Retrieval-Augmented Generation (RAG) pipeline that stores, retrieves, and utilizes past conversations to generate context-aware responses.

---

## 🚀 Features

- 🔍 Semantic search over past conversations
- 💾 Persistent memory using ChromaDB
- 🧠 Fast embeddings with SentenceTransformers
- ⚡ LLM-powered responses via Groq (LLaMA 3)
- 🧩 Modular, clean, and extensible architecture
